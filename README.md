# GlobalAdSlots

Static prototype for **GLOBAL ADSLOTS**:

> The Global Marketplace for Advertising Inventory & Last-Minute Media Deals

## Run locally

```bash
cd GlobalAdSlots
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
